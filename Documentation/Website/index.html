drawCircle function

The drawCircle function draws a circle with a specified center point, radius, and color on a pixel frame buffer.

Function Signature

function drawCircle(x, y, radius, color) {
    // implementation
}

Parameters

  • x (number): An integer representing the x coordinate of the center of the circle.
  • y (number): An integer representing the y coordinate of the center of the circle.
  • radius (number): An integer representing the radius of the circle.
  • color (array): An array of three integers representing the RGB color values for the circle.

Return Value

The drawCircle function does not return any value.

drawText function

The drawText function draws text with a specified color at a specified location on a pixel frame buffer.

Function Signature

function drawText(text, x, y, color) {
    // implementation
}

Parameters

  • text (string): A string representing the text to be drawn on the pixel frame buffer.
  • x (number): An integer representing the x coordinate of the top-left corner of the text.
  • y (number): An integer representing the y coordinate of the top-left corner of the text.
  • color (array): An array of three integers representing the RGB color values for the text.

Return Value

The drawText function does not return any value.

scrollText function

The scrollText function animates text scrolling from right to left on a pixel frame buffer.

Function Signature

function scrollText(endX, y, text, color, waitTime) {
    // implementation
}

Parameters

  • endX (number): An integer representing the x coordinate where the scrolling text will stop.
  • y (number): An integer representing the y coordinate of the top-left corner of the scrolling text.
  • text (string): A string representing the text to be scrolled on the pixel frame buffer.
  • color (array): An array of three integers representing the RGB color values for the scrolling text.
  • waitTime (number): A floating-point number representing the duration in seconds between each frame of the scrolling text animation.

Return Value

The scrollText function does not return any value.

displayImage function

The displayImage function displays an image on a pixel frame buffer.

Function Signature

function displayImage(imagePath, blend) {
    // implementation
}

Parameters

  • imagePath (string): A string representing the file path of the image to be displayed on the pixel frame buffer.
